# 一、完整数据库表设计

## 2.1 用户与认证模块 (User Module)

### 表名：users (用户表)

| 字段名        | 类型         | 必填 | 说明                          | 对应功能 |
| ------------- | ------------ | ---- | ----------------------------- | -------- |
| id            | BIGINT       | YES  | 主键，自增                    | -        |
| username      | VARCHAR(50)  | YES  | 唯一，用户名/手机号           | U01, U02 |
| email         | VARCHAR(100) | YES  | 唯一，邮箱                    | U01      |
| password_hash | VARCHAR(255) | YES  | BCrypt 加密后的密码           | U01      |
| nickname      | VARCHAR(50)  | NO   | 昵称                          | U03      |
| avatar_url    | VARCHAR(255) | NO   | 头像 OSS 地址                 | U03      |
| student_id    | VARCHAR(20)  | NO   | 学号                          | U03      |
| school        | VARCHAR(50)  | NO   | 学院名称                      | U03      |
| bio           | VARCHAR(200) | NO   | 个性签名                      | U03      |
| points        | INT          | YES  | 当前积分余额，默认为 0        | U04      |
| role          | VARCHAR(20)  | YES  | USER 或 ADMIN                 | 权限控制 |
| version       | INT          | YES  | 乐观锁版本号 (用于并发扣积分) | U04      |
| created_at    | DATETIME     | YES  | 注册时间                      | -        |
| updated_at    | DATETIME     | YES  | 更新时间                      | -        |

### 表名：point_transactions (积分流水表)

| 字段名      | 类型         | 说明                                                                     | 对应功能 |
| ----------- | ------------ | ------------------------------------------------------------------------ | -------- |
| id          | BIGINT       | 主键                                                                     | -        |
| user_id     | BIGINT       | 关联用户 ID                                                              | -        |
| amount      | INT          | 变动数值 (如: +10, -50)                                                  | U04      |
| type        | TINYINT      | 类型: 1-签到, 2-发帖奖励, 3-评论奖励, 4-置顶消耗, 5-跑腿支付, 6-跑腿收入 | U04      |
| description | VARCHAR(100) | 描述 (如 "2023-10-01 每日登录")                                          | -        |
| created_at  | DATETIME     | 发生时间                                                                 | -        |

## 2.2 社交核心模块 (Social Module)

### **表名：categories (分类表)**

| 字段名     | 类型         | 必填 | 说明                        | 对应功能 |
| ---------- | ------------ | ---- | --------------------------- | -------- |
| id         | BIGINT       | YES  | 主键，自增                  | -        |
| name       | VARCHAR(50)  | YES  | 分类名称，如 "闲置"、"吐槽" | S02      |
| icon       | VARCHAR(255) | NO   | 分类图标 URL                | S02      |
| sort_order | INT          | YES  | 排序权重，默认 0            | S02      |
| is_active  | BOOLEAN      | YES  | 是否启用，默认 TRUE         | -        |
| created_at | DATETIME     | YES  | 创建时间                    | -        |

### 表名：posts (帖子表)

| 字段名        | 类型         | 说明                                              | 对应功能 |
| ------------- | ------------ | ------------------------------------------------- | -------- |
| id            | BIGINT       | 主键                                              | -        |
| user_id       | BIGINT       | 发帖人 ID                                         | S01      |
| category_id   | BIGINT       | 分类 ID，关联 categories.id                       | S01, S02 |
| is_anonymous  | BOOLEAN      | 是否匿名发布，默认 FALSE                          | S01      |
| content       | TEXT         | 帖子文本内容                                      | S01      |
| summary       | VARCHAR(500) | AI 生成的摘要 (MVP 可选字段)                      | I03      |
| images        | JSON         | 图片 URL 数组 (如 ["url1", "url2"])               | S01      |
| tags          | VARCHAR(255) | 标签字符串，用逗号分隔 (如 "吐槽,求助")           | S01, S02 |
| product_info  | JSON         | 商品信息，如 {"price": 980}，闲置帖专用           | S01      |
| view_count    | INT          | 浏览量                                            | S02      |
| like_count    | INT          | 点赞数 (用于数据持久化，高并发下可由 Redis 同步)  | S04, N02 |
| comment_count | INT          | 评论数                                            | S05, N02 |
| collect_count | INT          | 收藏数，默认 0                                    | S11      |
| hot_score     | INT          | 热度分数，定时计算更新，默认 0                    | S03      |
| is_sticky     | BOOLEAN      | 是否置顶 (0:否, 1:是)                             | S07      |
| status        | TINYINT      | 状态: 0-正常, 1-被举报(待审), 2-已删除(违规/自删) | S06, S07 |
| created_at    | DATETIME     | 发布时间                                          | S02      |

### 表名：comments (评论表)

| 字段名           | 类型         | 说明                                    | 对应功能 |
| ---------------- | ------------ | --------------------------------------- | -------- |
| id               | BIGINT       | 主键                                    | -        |
| post_id          | BIGINT       | 关联帖子 ID                             | S05      |
| user_id          | BIGINT       | 评论人 ID                               | S05      |
| parent_id        | BIGINT       | 父评论 ID (若为 0 或 NULL 则为一级评论) | S05      |
| reply_to_user_id | BIGINT       | 被回复者用户 ID（楼中楼回复某人时使用） | S05, S12 |
| content          | VARCHAR(500) | 评论内容                                | S05      |
| like_count       | INT          | 评论点赞数，默认 0                      | S13      |
| status           | TINYINT      | 状态: 0-正常, 1-已删除，默认 0          | S14      |
| created_at       | DATETIME     | 评论时间                                | -        |

### 表名：post_likes (帖子点赞关联表)

| 字段名     | 类型     | 说明                            | 对应功能     |
| ---------- | -------- | ------------------------------- | ------------ |
| id         | BIGINT   | 主键                            | -            |
| post_id    | BIGINT   | 帖子 ID                         | S04          |
| user_id    | BIGINT   | 用户 ID                         | S04          |
| created_at | DATETIME | 点赞时间                        | -            |
| 索引       | UNIQUE   | (post_id, user_id) 联合唯一索引 | 防止重复点赞 |

### 表名：comment_likes (评论点赞关联表)

| 字段名     | 类型     | 说明                               | 对应功能     |
| ---------- | -------- | ---------------------------------- | ------------ |
| id         | BIGINT   | 主键                               | -            |
| comment_id | BIGINT   | 评论 ID                            | S13          |
| user_id    | BIGINT   | 用户 ID                            | S13          |
| created_at | DATETIME | 点赞时间                           | -            |
| 索引       | UNIQUE   | (comment_id, user_id) 联合唯一索引 | 防止重复点赞 |

### 表名：post_collections (收藏表)

| 字段名     | 类型     | 说明                            | 对应功能       |
| ---------- | -------- | ------------------------------- | -------------- |
| id         | BIGINT   | 主键，自增                      | -              |
| user_id    | BIGINT   | 收藏人 ID (关联 users.id)       | 查看"我的收藏" |
| post_id    | BIGINT   | 帖子 ID (关联 posts.id)         | 跳转到详情页   |
| created_at | DATETIME | 收藏时间                        | 按时间排序     |
| 索引       | UNIQUE   | (user_id, post_id) 联合唯一索引 | 防止重复收藏   |

### 表名：user_follows (关注表)

| 字段名       | 类型     | 必填                                     | 说明                                | 对应功能           |
| ------------ | -------- | ---------------------------------------- | ----------------------------------- | ------------------ |
| id           | BIGINT   | YES                                      | 主键，自增                          | -                  |
| follower_id  | BIGINT   | YES                                      | 发起关注者 ID (粉丝)，关联 users.id | 查看"我的关注列表" |
| following_id | BIGINT   | YES                                      | 被关注者 ID (目标)，关联 users.id   | 查看"我的粉丝列表" |
| created_at   | DATETIME | YES                                      | 关注时间                            | 按关注时间排序     |
| 索引         | UNIQUE   | (follower_id, following_id) 联合唯一索引 | 防止重复关注                        |                    |

### 表名：reports (举报记录表)

| 字段名      | 类型        | 说明                                               | 对应功能 |
| ----------- | ----------- | -------------------------------------------------- | -------- |
| id          | BIGINT      | 主键                                               | -        |
| target_id   | BIGINT      | 被举报对象 ID (Post ID 或 Comment ID)              | S06      |
| target_type | TINYINT     | 类型: 1-帖子, 2-评论                               | S06      |
| reporter_id | BIGINT      | 举报人 ID                                          | S06      |
| reason      | VARCHAR(50) | 举报理由 (如: 垃圾广告, 攻击辱骂)                  | S06      |
| status      | TINYINT     | 处理状态: 0-待处理, 1-已处理(违规), 2-已驳回(误报) | S07      |
| created_at  | DATETIME    | 举报时间                                           | -        |

## 2.3 创新特色模块 - 校园跑腿 (Errands Module)

### 表名：errands (跑腿订单表)

| 字段名        | 类型          | 说明                                                   | 对应功能 |
| ------------- | ------------- | ------------------------------------------------------ | -------- |
| id            | BIGINT        | 主键                                                   | -        |
| publisher_id  | BIGINT        | 发布者 ID                                              | I01      |
| runner_id     | BIGINT        | 接单人 ID (NULL 表示无人接单)                          | I01      |
| content       | VARCHAR(255)  | 需求描述 (取快递/买饭)                                 | I01      |
| pickup_addr   | VARCHAR(100)  | 取货地点                                               | I01      |
| delivery_addr | VARCHAR(100)  | 送达地点                                               | I01      |
| hidden_info   | VARCHAR(255)  | 隐私信息 (如取件码)，仅接单人/发布者可见               | I01      |
| bounty        | DECIMAL(10,2) | 悬赏金额/积分数                                        | I01      |
| currency      | TINYINT       | 货币类型: 1-积分, 2-现金                               | I01      |
| deadline      | DATETIME      | 截止时间                                               | I01      |
| status        | TINYINT       | 状态: 0-待接单, 1-进行中, 2-已完成, 3-已取消, 4-已过期 | I01      |
| version       | INT           | 乐观锁版本号 (防止多人同时抢单)                        | I01      |
| created_at    | DATETIME      | 发布时间                                               | -        |
| completed_at  | DATETIME      | 完成时间                                               | -        |

## 2.4 消息与通知模块 (Notification Module)

### 表名：conversations (会话表)

| 字段名            | 类型     | 说明                              | 对应功能     |
| ----------------- | -------- | --------------------------------- | ------------ |
| id                | BIGINT   | 主键，自增                        | N04          |
| user1_id          | BIGINT   | 用户 A ID（较小的 ID）            | N04          |
| user2_id          | BIGINT   | 用户 B ID（较大的 ID）            | N04          |
| last_message_id   | BIGINT   | 最后一条消息 ID                   | N04          |
| last_message_time | DATETIME | 最后消息时间（便于排序）          | N04          |
| user1_unread      | INT      | 用户 A 未读消息数，默认 0         | N04          |
| user2_unread      | INT      | 用户 B 未读消息数，默认 0         | N04          |
| created_at        | DATETIME | 创建时间                          | -            |
| 索引              | UNIQUE   | (user1_id, user2_id) 联合唯一索引 | 确保会话唯一 |

### 表名：chat_messages (私聊消息表)

| 字段名          | 类型        | 说明                                       | 对应功能 |
| --------------- | ----------- | ------------------------------------------ | -------- |
| id              | BIGINT      | 主键                                       | -        |
| conversation_id | BIGINT      | 会话 ID，关联 conversations.id             | N04      |
| sender_id       | BIGINT      | 发送者 ID                                  | N04      |
| receiver_id     | BIGINT      | 接收者 ID                                  | N04      |
| content         | TEXT        | 消息内容                                   | N04      |
| msg_type        | TINYINT     | 类型: 0-文本, 1-图片                       | N04      |
| status          | VARCHAR(20) | 消息状态: sent/delivered/read，默认 'sent' | N04      |
| is_read         | BOOLEAN     | 是否已读 (0:未读, 1:已读)                  | N04      |
| created_at      | DATETIME    | 发送时间                                   | -        |

### 表名：notifications (系统通知表)

| 字段名      | 类型         | 说明                                                                     | 对应功能 |
| ----------- | ------------ | ------------------------------------------------------------------------ | -------- |
| id          | BIGINT       | 主键                                                                     | -        |
| user_id     | BIGINT       | 接收通知的用户 ID                                                        | N03      |
| sender_id   | BIGINT       | 触发动作的用户 ID (如点赞人)                                             | N03      |
| type        | TINYINT      | 1-点赞(like), 2-评论(comment), 3-回复(reply), 4-关注(follow), 5-系统通知 | N03      |
| post_id     | BIGINT       | 关联帖子 ID（可为 NULL）                                                 | N03      |
| comment_id  | BIGINT       | 关联评论 ID（可为 NULL）                                                 | N03      |
| reply_id    | BIGINT       | 关联回复 ID（可为 NULL）                                                 | N03      |
| quote       | VARCHAR(255) | 引用内容摘要（帖子/评论的部分内容）                                      | N03      |
| quote_label | VARCHAR(50)  | 引用标签，如 "我的帖子"、"我的评论"                                      | N03      |
| content     | VARCHAR(255) | 通知摘要 (如: "张三评论了你的帖子")                                      | N03      |
| is_read     | BOOLEAN      | 是否已读                                                                 | N03      |
| created_at  | DATETIME     | 时间                                                                     | -        |
